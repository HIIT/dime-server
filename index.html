Digital Me

The task of the Digital Me (DiMe) server is to collect your personal data from various loggers into a central place that you control. You can run DiMe in your local machine or in a server. The DiMe server source code is open source software licensed under the MIT License.

You can start collecting your personal data in two easy steps:

  1. Install the DiMe core server on your own machine. See the README on github for instructions.
  2. Install one or more loggers which collect different aspects of your life and log them to your own DiMe server.

    Here will be a list of loggers soon :-)

The Digital Me software is being developed within the Re:Know research project, which aims to create information-seeking and sense-making methods supporting a revolution of knowledge work.
